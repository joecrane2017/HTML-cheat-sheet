HTML Tables

Tags

Tag Name Description
<table> Table The wrapper element for HTML tables.
<thead> Table Head The elements that wrap the rows that will be the column headers.
<tbody> Table Body The elements that wrap the rows that make up the main body of the table.
<tr> Table Row The container for the table data that makes up an individual row.
<td> Table Data The container for the actual table data in a single cell of a table row.
<tfoot> Table Footer The elements that wrap the footer row of a table. (Often wrapping a summary row that summarizes each column)

Attributes

Attribute Name Description
<colspan> Column Span Defines how many columns across a row should span.
<rowspan> Row Span Defines how many rows down a column should span.
